ProjetIA
==========

Compilation
-----------
 * make
 
Exécution
---------

 * ./main
 * ./main pc pm nb_generations


	pc -> probabilité de croisement
	pm -> probabilité de mutation
	nb_generations -> nombre max de générations

	valeurs par défaut si les arguments ne sont pas fournis :
	pc = 0.85
	pm = 0.1
	nb_generations = 1000
